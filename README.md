# Short Cirkuit Reskin

## Estructura

- `backend/` → Node + Prisma + Mongo
- `frontend/` → React 

## Backend

```bash
cd backend
cp .env.example .env  
npm install
npm run db:setup               
npm run dev            
npm run build
npm run start
```

## Frontend
```bash
cd frontend
cp .env.example .env   
npm install
npm run dev            
npm run build          
```
